How It Works

4-Stage AI Pipeline

Each agent hands off to the next. No manual intervention required.

1

MarketData Agent

Fetches 1–10 years of OHLCV data via yfinance, computes 9 indicators (SMA, EMA, RSI, MACD, Bollinger, ATR…), and produces a market summary.

2

Strategy Agent

GPT-4o reads the market summary and generates a complete backtesting.py Strategy class with stop-loss, entry/exit logic, and tunable parameters.

3

Python Master Agent

AST-level static analysis + runtime dry-run + GPT-4o repair loop. Catches look-ahead bias, invalid imports, leverage violations, and more.

4

Backtest + Critique Agent

Sandboxed execution, 10+ metrics, walk-forward validation, then a self-critique agent generates a revised improved strategy automatically.
